## Run Locally

Clone the project

```bash
  git clone https://github.com/thurdelima/nestjs-jwt-users
```

Go to the project directory

```bash
  cd nestjs-jwt-users
```

Install dependencies

```bash
  yarn install 
```

Configure .ENV vars

And

```bash
  npm run start:dev

```
